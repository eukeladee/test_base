## ASCII To Text Converter - ATT -

### Installation & Usage

To use this program just run this commands in your terminal.

`$ git clone https://github.com/eukeladee/test_base.git`

`$ cd test_base/src/`

`$ chmod u+x ./compile.sh`

`$ ./att "text"`
